[Paper Summary] Understanding Personality through Social Media

Posted on 2017/04/05 - 三 in Paper Summary

Paper

Main Purpose: How linguistic features correlate with each personliaty trait.

Use Twitter to predict MBIT personality.

Problem of past researches

  • Word usage in social media might not be formal which makes typical linguistic analysis tool perform poorly
  • Gain personliaty information is costly (e.g. Big Five Questionnaries)

MBTI

Instead of commonly used big five theory, MBTI is used to identify personliaty

Myers-Briggs Type Indicator

There are 4 types of personliaty
 i.e.

  • Introversion(I) / Extraversion(E)
  • Intuition(N) / Sensing(S)
  • Feeling(F) / Thi nking(T)
  • Perception(P) / Judging(J)

In each type, one value is chosen.
Thus, personliaty can be expressed as a code with 4 letters.
e.g. ENFJ

Data

  • 90,000 users

  • personliaty-related tweets from 2006~2015

  • Heuristic rules.
     e.g.

  • I’m
  • I got
  • I have been a
  • No complicated classifcation method is used for certainty

120K out of all 1.7M tweetw

filter out tweets with personality codes

personliaty distribution is skewed

Features

  • n-grams
    • most frequent 1000
    • Every user with a 3000 dimensions
  • Twitter POS tags
    • 25 types
  • word vectors
    • 2,334,564 words
    • 500 dimension
    • Average word vectors
    • Weighted average word vectors (TF-IDF)

Prediction

Logistic Regression
but Random Forest and SVM produced similar results

we aim to find the linguistic features that distinguish people with different personality types and explore how these features can be explain by personality
