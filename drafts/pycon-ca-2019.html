PyCon CA 2019

隔了一個多月，終於有時間可以來補一下 PyCon CA 2019 的心得
這是我今年參加的第四場 PyCon
終於是因為當 Regular Talk 的講者來參加

當初會想投稿的原因，就單純的是被 PyCon CA 的 Organizer 在 PyCon US 的閃電秀燒到
「秋天剛好是來加拿大看楓葉的好時間，來投稿吧」
就覺得好像還沒來過加拿大就來投稿看看吧 XD
雖然這麼說，但這個想法也是擱置了好一段時間
之後開始比較認真在構思，則是因為有其他想去北美一趟的理由

想想之後也會協助 PyCon TW 2020 議程組
來當講者看看，好像也會知道大家都在幹嘛（？？？

講者宣傳

他們每一位講者的 talk 都會發篇推特宣傳
而且會積極地邀請你一起來宣傳自己的 talk
我覺得這點還蠻有趣的

會場

PyCon CA 辦在一個市中心的宴會廳

Keynote 的演講廳完全是大家可以邊吃飯邊聽 talk 的 XD

議程

提醒議程的方式很好玩
在走廊會閃燈，提醒大家議程即將開始
原本以為是劇場的三明三暗
但好像都會閃超過三次
不過我覺得這個做法蠻有趣的

第一天早上去當志工幫忙指路，就錯過了 Keynote

第一場我聽的是 Understanding autistic children using BioSensors and Python!
稍微詳細一點的筆記我都丟在 note
聽到最後才發現這是講者的大學專題
大致上是在講如何透過 sensor 來偵測自閉症
實際有問過一些醫師要怎麼做 feature engineering

再來去聽了 My Favourite Errors - Tales From The Ops Side
note
主要在講各種可能遇到的 HTTP 錯誤
這場應該是我覺得 PyCon CA 數一數二精彩的 talk
雖然跟 Python 完全沒有直接關係 XD
後來發現講者前一段時間才在 devops day 講過同樣的講題

接下來連三場，我跑去主持人，就沒特別花時間做筆記
畢竟用非母語主持其實還是有些吃力
主持完真的太累，就一口氣休息了幾場 * Modelling user journeys with Keras and neural networks
* 第一場主持的太緊張了，沒有什麼聽到 XD * Anomaly detection in the wild
* 這場在講如何偵測異常值，我覺得還蠻好玩的 * A Pythonista’s intro to Kafka: no, it’s not like Celery
* 如果對 Kafka 有興趣，這場對 Kafka 的介紹還蠻不錯的，值得一聽

第二天早上是閃電秀
登記閃電秀和其他活動的方式就是直接上去改他們的 wiki 吧
pyconca/2019-wiki
我也投了一個小小的閃電秀來宣傳一下明年的 PyCon TW
迴響還蠻不錯的
PyCon TW 2020
倒是這天因為太冷了，我有點睡過頭，差點就趕不上閃電秀...
我完全是到現場馬上就衝上台

因為我是這天的下午給 talk，聽的議程就比較少
印象最深刻的是 Fun with compilers: exploring languages one Python at a time
GitHub
之前好像有人問我 Python 到底是編譯還是直譯
這位講者給的答案是 Python 就只是個語言，要看用哪個 implementation 來跑

這些是之後影片出來我會想要補的議程

Python Table Manners - A Clean Style

slide

我的分享主要是整理了一大堆工具，他們可能能在開發的各個不同環節用上
我把這次的分享定位在「讓你知道有這個工具，我怎麼用它，你如果會用到，可以自己研究細節」
因為介紹了很多工具，所以沒有辦法深入討論
當初也有點擔心這樣的內容會不會太淺
出發去 PyCon CA 前，有先到 Taipei.py 分享一下
得到的迴響比我預想的還好些
PyCon CA 當天來聽的人也算不少
應該還算成功吧（？

一個有趣的小插曲是
我原本介紹了一套 Java Script 的工具 commitizen
雖然 Python 也有自己的 commitizen
但一開始用的時候，遇到了很多的 bug
PyCon CA 前一個禮拜，就覺得還是想介紹個 Python 的貢獻
拼拼看能不能在分享前把這些 Bug 修完
認真使用才發現功能大部分都沒問題，只是我剛好都遇到 Bug......
幸運的是 commitizen 作者回復超快
大概在 PyCon CA 前，我就把我遇到的 Bug 都修完了
既然都修完了，就順便把這段的投影片也更新一下
後來還拿著這個專案去主持一個 Sprint

Development Sprint

參與的專案還蠻多的
在 PyCon CA 前，幾乎沒有人想填 wiki
都是到 PyCon CA 開始後，甚至是 Sprint 開始後才慢慢有專案出現 XD

原則上不供餐，鼓勵你和你的夥伴一起出去吃飯
其實我覺得挺好的
雖然第二天還是訂了 Pizza XD

整體來說

整體來說，PyCon CA 給我的感覺是大家的距離是很近的
大家就是一群來這裡的 Pythonista
不會因為你有不同的身份，而有點小隔閡的感覺
也有可能是因為我當了志工才有這樣的感覺

可能因為場地也不是很大
就小小一層
但也不是說他們不盛大

紀念品還蠻有趣的，很符合當地的毛帽
有一度想要買
因為真的很冷...
而且我在加拿大期間最冷的就是這兩天
-8 度真的瘋了
但又不能不出門......
